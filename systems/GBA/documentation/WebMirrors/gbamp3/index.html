GBAmp3

About

GBAmp is a little but powerful amplifier module made to fit your favourite Game Boy.
Most Game Boys has pretty low spekaer volume, but GBAmp can make it sound so that your neighbours can hear it :) GBAmp is a "Hi-Fi" grade Class-D amplifier which is more power efficient than Game Boy internal amplifier. This can help your batteries last longer.
You can also read the full GBAmp story.

New in version 3

Mute speaker when headphones inserted. Most requested feature!
Better fit for Game Boy Color if using original speaker.
Game Boy Advance supported.
It's cool black now!

How to Install

Select your Game Boy model to get installation guide

  • Game Boy DMG
  • Game Boy Color GBC
  • Game Boy Pocket GBP
  • Game Boy Advance GBA

You will need

  • 8 wires of AWG 28-30
  • Basic soldering iron and some solder
  • Skills
Follow wires guide picture to see where to solder each wire. Also reference to installed GBAmp picture to see where to put wires and module.

  • 8 wires of AWG 28-30
  • Basic soldering iron and some solder
  • New slim speaker (but original works too) for better fit and loudness
  • Skills
  • In case you do not have EM4 component on the pcb, solder green wire to the green circled pad.
Follow wires guide picture to see where to solder each wire. Also reference to installed GBAmp picture to see where to put wires and module.
